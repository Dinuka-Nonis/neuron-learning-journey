# Neuron Learning Journey

Learning differentiable programming through computational neuroscience.

## Goal
Build a neuron simulator step-by-step, understanding each layer:
1. Define what exists (data structures)
2. Forward simulation (physics)
3. Error measurement (loss)
4. Gradients (optimization)
5. Learning (parameter fitting)

## Structure
- `src/` - Core simulation code
- `notebooks/` - Learning experiments
- `docs/` - Notes and explanations
